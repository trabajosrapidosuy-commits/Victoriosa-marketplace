# Victoriosa Supplier Policy

## Criterios para aceptar proveedor
- Identidad y URL verificable.
- Permite reventa o afiliacion.
- Permite uso de imagenes o provee material autorizado.
- Politica de devoluciones visible.
- Tiempos de envio razonables.
- Sin historial evidente de falsificaciones.
- Productos compatibles con belleza/wellness no medico.

## Criterios para bloquear proveedor
- Replica, fake, 1:1 o marcas no autorizadas.
- Productos peligrosos o restringidos.
- Imagenes sin permiso.
- Reclamos repetidos.
- Envio demasiado largo sin aviso.
- Margen negativo o costos ocultos.

## Productos de alto riesgo
Cosmetica sin ingredientes o proveedor claro, electronicos sin certificacion, productos infantiles, claims medicos, suplementos o productos ingeribles.

## Politica de imagenes
No copiar imagenes de terceros sin permiso. Usar placeholder, imagen propia, catalogo autorizado o status `needs_review`.
